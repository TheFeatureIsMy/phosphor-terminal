总览驾驶舱

总览页是 AlphaLoop 的生产驾驶舱。所有指标来自后端真实 API：账户、策略、持仓、风控、决策、告警、Provider 健康、AI 模型、最新信号。顶部 bar 显示当前模式与全平台状态。无占位数值。

Top Bar（顶部栏）

Top Bar 拆为两行，覆盖全应用：

行 1 — 品牌 + 模式 + 主动作

  • 品牌：左侧 "弈机 AlphaLoop"（仅 Dashboard 路由显示，其它路由显示面包屑）
  • 模式胶裹（ModePill）：实时显示 LIVE / PAPER / DRYRUN / STOPPED / MOCK / NOT READY 之一，颜色随状态变化
  • 主动作：语言切换 / 主题切换 / 命令面板 (⌘K) / 通知（未读红点）/ 紧急停止（直接调用 APIEmergency）

行 2 — 状态描点条（DashboardStatusStrip）

7 个状态 chip 横向排列：

  • PROVIDERS — 显示 healthy/total 比例
  • EXCHANGE — OK / WARNING / DOWN
  • REDIS — RTT 毫秒
  • FREQTRADE — HEALTHY / WARNING / DOWN
  • RISK — NORMAL / WARNING / LOCKED
  • POSITIONS — 当前持仓数
  • LAST UPDATE — 上次刷新时间（15s 轮询）

Page Header（页面标题）

Dashboard 顶部固定显示：

  • // 驾驶舱 COCKPIT — 单页标题（避免与 Top Bar 重复）
  • 数据源徽章：LIVE（绿）/ DATA SOURCE UNAVAILABLE（黄）
  • 副标题：实时驾驶舱 · 所有指标均为后端真实数据

卡片布局（Bento 网格）

DashboardStatusBar — 基础设施状态条

单行 4 个单元：系统 / Freqtrade 延迟 / Redis 延迟 / 交易所。后端 BFF system 字段。

AccountHeroCard — 账户总览（Hero）

左：大字权益（带数字滚动动画）+ 24h 涨跌幅 + 模式胶裹 + sparkline（来自 /api/dashboard/equity-curve）。

右：5 列 KPI — 今日盈亏 / 本周盈亏 / 累计盈亏（来自 KPIs）/ 胜率（来自 KPIs）/ 夏普比率。权益 = BFF `account.equity`；双源均为 0 时显示 — + DATA SOURCE UNAVAILABLE 徽章。

AvailableActionsRow — 建议操作

系统根据 BFF `availableActions` 显示最多 3 个按钮。点击走 viewModel.performAction(_:)：紧急停止直接调用 APIEmergency.emergencyStop；启动实盘 / 取消挂单等跳转到对应页面。

三列指标卡（Runtime / Readiness / Risk）

  • StrategyRuntimeCard — 运行中策略数 + 持仓 / 挂单 / 对账 三色描点
  • LiveReadinessCard — 准入灯（实盘就绪 / 仅模拟 / 风控锁定 / 紧急锁定）+ 真实 checks 列表（不再硬塞 7 个 OK chip）
  • GlobalRiskCard — 全局风控状态药丸 + 日损余额% / 周损余额% 进度条

PositionRiskTable — 持仓风险表

全宽表格。数据来自 GET /api/execution/positions（已删除硬编码 mockPositions）。列：品种 / 方向（左色条）/ 数量 / 开仓价 / 现价 / 盈亏 / 盈亏% / 状态徽章（本端/交易所同步 / 存在差异 / 仅本端 / 仅交易所）。无持仓时显示 无持仓；后端不可达时显示 DATA SOURCE UNAVAILABLE。

ProviderHealthCard + AIModelStatusCard

两列网格，分别显示：

  • Provider：healthy / warning / error 三类计数 + per-entry 展开（名称、延迟、最近错误）
  • AI 模型：每个模型 provider / state / GPU 内存

SignalsFeedCard — 信号流（含溯源）

来自 GET /api/agent-signals/signals。每个信号显式展示 3 个溯源标签：AGENT / STRATEGY / SNAPSHOT（未关联时显示 NO TRACE）。

RecentDecisionFeed + AlertTimeline

左右双栏。BFF recent_decisions 与 alerts，数据为空时分别显示 暂无决策 / 暂无告警。

EmergencyActionBar — 紧急控制

底部固定：图标 + 描述 + 红色「停止全部交易」按钮。点击 → 二次确认 → 真实 APIEmergency.emergencyStop。

LearnAlphaLoopCard — 学习卡

仅当 state == "unknown" 且无任何数据时显示，引导用户打开用户指南。可手动关闭。

数据刷新

页面级 30s 轮询。Top bar 状态条 15s 轮询。紧急按钮点击 → 立即 APIEmergency.emergencyStop → toast 反馈 → 重新 load()。

数据源失败回退

任一卡片依赖的子端点失败：

  • 该卡片渲染空态 + 标签，其它卡片不受影响
  • 整体 BFF state == "data_source_unavailable" → 渲染 DataSourceUnavailableView（图标 + 标题 + 重试按钮）
