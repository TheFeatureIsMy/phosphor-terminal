总览仪表盘

仪表盘是 AlphaLoop 的交易运营总控台，一屏展示账户、策略、持仓、风控、决策和告警全貌。

布局结构

仪表盘采用 Bento 网格布局，从上到下依次为：

全局状态栏

顶部固定的基础设施状态栏，展示系统状态、Freqtrade 延迟、Redis 延迟、交易所连接状态，以及系统级原因码标签。

账户总览（Hero 卡片）

左侧显示总权益（大字带动画）和24小时涨跌幅，右侧四列指标：今日盈亏、本周盈亏、最大回撤、夏普比率。

建议操作

系统根据当前状态推荐的快捷操作按钮，如部署策略、审核信号、收紧止损等。最多显示3个。

三列指标卡

  • 策略运行 — 运行中策略数、持仓数、挂单数、对账数
  • 实盘准入 — 当前准入状态灯（实盘就绪/仅模拟/风控锁定）及检查通过情况
  • 全局风控 — 风控状态、日损余额%、周损余额%进度条

持仓风险表

全宽表格展示当前持仓：品种、方向（左侧绿/红色条）、数量、开仓价、盈亏、风险等级和原因码。

决策流 + 告警时间线

左右双栏：

  • 近期决策 — 最近的交易决策，显示时间、品种、操作（执行/持有/减仓/拒绝）及原因码
  • 告警时间线 — 按时间排列的系统和交易告警（信息/警告/错误三级）

紧急停止栏

底部固定的紧急控制栏，包含"停止全部交易"按钮。点击后需确认，确认后将停止所有策略并取消挂单。

数据刷新

仪表盘每30秒自动轮询刷新数据。所有数据来自单一 BFF 聚合接口，确保一致性。
