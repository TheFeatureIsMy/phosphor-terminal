策略工作区

页面 · 策略 · 5 分钟

集中管理所有量化策略的地方 — 创建、浏览、重命名、删除，然后进入详情页进行深度编辑。

TL;DR

策略工作区是进入 AlphaLoop 的第一步。左侧是策略列表（卡片网格），每张卡片显示策略名称、状态（草稿/运行中/已暂停）、类型和来源。点击卡片进入详情页；右上角"新建策略"按钮打开创建面板。创建后自动跳转到详情页，你可以在那里编写 DSL 规则、打开画布、运行回测或启动模拟。

谁用 · 何时用

每个用户都会从这里开始。首次使用时，列表为空，你会看到引导文案和一个"新建策略"按钮。之后每次回到 AlphaLoop，策略工作区是你的"基地"——在这里选择要编辑的策略，或者清理不再需要的旧策略。策略数量较多时，顶部的状态统计（总计/草稿/运行中）可以帮你快速了解全局。

页面解剖

  1. ① 策略卡片网格 — 每张卡片展示策略名称、状态指示灯（绿色=运行中、黄色=草稿、灰色=已归档）、类型标签和简短描述。右键打开上下文菜单可重命名或删除。点击卡片进入详情页。
  2. ② 状态统计栏 — 顶部显示总计、草稿、运行中三个计数徽章。一眼掌握策略组合整体状态。
  3. ③ 新建策略面板 — 点击右上角按钮后从顶部滑入的覆层面板。输入策略名称，点击创建即跳转到详情页。面板采用液态玻璃背景，点击外部可关闭。
  4. ④ 详情页导航 — 创建或点击卡片后进入策略详情页，在那里你可以看到概览、DSL 规则、画布、回测、版本、信号、模拟、风控等 10 个标签页。

关键指标怎么读

  • 状态指示灯 — 绿色=运行中（策略已连接交易所并持续执行），黄色=草稿（仍在编辑，未上线），橙色=已暂停，灰色=已归档（不再使用）。
  • 策略类型 — 显示策略的算法类别（趋势跟踪、均值回归、网格、套利等），帮助快速筛选。
  • 来源 — 表明策略是如何创建的：自定义（手动编写）、模板（从模板库选用）或 AI 生成。

常见操作 · 一步步

  1. 创建新策略 — 点击右上角"新建策略"按钮，输入名称（如"趋势跟踪 v4"），点击"创建策略"。创建后自动进入详情页。
  2. 进入策略详情 — 在卡片网格中点击任意策略卡片，跳转到详情页。详情页顶部显示策略名称、状态、类型和来源，下方是 10 个功能标签页。
  3. 重命名策略 — 右键点击策略卡片，选择"重命名"，在弹出窗口中输入新名称并保存。
  4. 删除策略 — 右键点击策略卡片，选择"删除"，在确认对话框中确认。删除不可撤销。

幕后

策略列表通过 APIStrategiesV2.list() 从后端加载，返回 StrategyV2 数组。创建操作调用 api.create(name:) 后端接口，成功后更新 AppState 中的路由状态，使页面自动跳转到详情页。状态统计（草稿/运行中计数）在前端通过 viewModel.strategies 数组的 filter 计算得出，无需额外请求。列表刷新由 .task { await viewModel.load() } 在视图出现时触发。
