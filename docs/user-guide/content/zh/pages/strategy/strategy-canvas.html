策略画布

页面 · 策略 · 5 分钟

用拖拽的方式可视化构建交易策略 — 不用写一行代码，也能搭建专业级的量化策略逻辑。

TL;DR

策略画布是一个基于 React Flow 的可视化 DAG（有向无环图）编辑器，嵌入在 macOS 应用的 WKWebView 中。左侧节点面板列出所有可用组件（信号输入、指标条件、过滤器、仓位管理、风控策略、执行输出等），拖拽到画布上，用连线连接它们，就构成了完整的策略逻辑。右侧可打开 DSL 代码预览面板，实时查看图形转化后的 JSON 规则。底部的验证状态栏告诉你当前策略是否通过校验。

谁用 · 何时用

适合视觉思维者 — 不想手写 DSL JSON 时，在画布上拖拽节点更直观。策略详情页的"画布"标签页进入当前策略的画布，也可以从侧边栏打开独立的画布页面，切换不同策略。模板库提供 RSI 均值回归、布林带突破、MACD 趋势跟踪等预制模板，一键加载到画布上作为起点。

页面解剖

  1. ① 节点面板（左侧） — 列出所有可用的策略组件：信号输入、指标条件、过滤器、仓位管理、风控策略、执行输出、MTF Guard、结构防御等。点击节点将其添加到画布，或拖拽到画布上。可按类别筛选或搜索。收藏的节点出现在顶部。
  2. ② 画布区域（中间） — 基于 React Flow 的无限画布。节点显示为彩色卡片，端口（圆形小点）在左右两侧。从一个节点的输出端口拖拽到另一个节点的输入端口建立连线。支持缩放、平移、多选（Cmd+点击）和键盘删除。右下角有小地图帮助导航。
  3. ③ 节点配置面板（右侧） — 点击画布上的节点后打开。显示该节点的核心参数（如 RSI 周期、阈值）、端口连线状态、高级选项（名称、备注、执行条件）。参数变更实时生效。
  4. ④ 底部状态栏 — 显示验证状态（通过/失败/未验证）和画布统计（节点数、连线数）。右侧有"DSL 预览"切换按钮。
  5. ⑤ DSL 预览面板（可选） — 点击"DSL 预览"后在右侧滑出，实时显示画布图形转化后的 JSON 策略规则。支持复制到剪贴板。

关键指标怎么读

  • 验证状态 — 绿色"验证通过"表示画布中的节点和连线构成完整的、可编译的策略逻辑。红色"验证失败"并显示错误数，需修复后保存。
  • 节点与连线数 — 底部状态栏显示节点和连线的数量。更复杂的策略通常节点更多，但好的策略应该是"足够简单"的。
  • 端口状态 — 在配置面板中，已连线的端口显示绿色圆点和对端节点名；未连线的显示灰色，必填端口标有红色星号。

常见操作 · 一步步

  1. 添加节点 — 在左侧节点面板中，点击（或拖拽）一个节点类型（如"指标条件"）。节点出现在画布中央区域。
  2. 连接节点 — 从节点的输出端口（右侧圆点）拖拽到另一个节点的输入端口（左侧圆点）。连线自动使用平滑路径风格。
  3. 配置节点参数 — 点击画布上的节点，右侧配置面板自动打开。调整参数（如 RSI 周期=14，阈值<30），变更立即反映在 DSL 预览中。
  4. 验证并保存 — 点击顶部"验证"按钮检查策略合法性。验证通过后点击"保存"将画布内容保存为策略的新版本。
  5. 加载模板 — 点击"模板库"，从预设模板中选择一个（如"RSI 均值回归"），模板的节点和连线自动加载到画布上。
  6. 查看 DSL 代码 — 点击底部"DSL 预览"按钮，右侧面板显示画布转换后的 JSON 规则。点击复制图标可复制到剪贴板。

幕后

画布在 macOS 应用中通过 WKWebView 嵌入一个 React 应用。该 React 应用使用 @xyflow/react 库实现 DAG 编辑器。节点和连线的变更通过 CanvasBridge（WKScriptMessageHandler）发送到 Swift 端。图形到 DSL 的转换由 graphToDsl() 函数在前端完成，将节点和边映射为 StrategyRuleDSL v2.5 格式的 JSON。保存操作通过 APICanvas.save() 写入后端，同时通过 APIStrategiesV2.saveVersion() 创建新的策略版本。版本历史可在详情页的"版本"标签页中查看。
