市场结构

页面 · 结构 · 5 分钟

将当前市场的"故事"可视化——你处在什么市场状态、价格如何走到这里、哪些区域和流动性池正在等待价格反应。

TL;DR

市场结构页把 SMC（Smart Money Concepts）分析浓缩为一个因果叙事面板。顶部显示市场状态（趋势/震荡/高波动/恐慌）和结构评分。往下是结构事件时间线（BoS、CHoCH、Sweep、FVG Fill），再往下是当前活跃的价格区域和流动性池。选一个交易对和一个时间周期，页面会告诉你价格结构的完整故事。

谁用 · 何时用

日常做 SMC 分析的交易者。每天开盘前或每个交易时段开始时打开，快速了解标的是否处于健康的趋势结构、是否有值得关注的失衡区域。也适合在出现大幅波动后查看——检查是否发生了结构反转（CHoCH）或流动性扫除。

页面解剖

  1. ① 标头部 — 通过 ⌘K 切换交易对，用分段按钮选择分析时间周期（5m / 15m / 1h / 4h）。
  2. ② 市场状态 + 仪表盘 — 一个大号文字告诉你当前市场状态（上升趋势、下降趋势、震荡、高波动、恐慌），配上叙事描述。旁边的三个仪表盘显示结构评分（0-100）、溢价/折价判断和活跃区域计数。
  3. ③ 结构事件时间线 — 从旧到新排列的近期结构事件，每个事件用圆点+连线表示。类型包括：BoS（突破结构，趋势延续）、CHoCH（结构反转，趋势可能转向）、Sweep（流动性扫除）、FVG Fill（缺口回补）。
  4. ④ 活跃区域 — 价格尚未完全反应的 SMC 区域，按距离排序。每个区域显示与现价的百分比距离、区域类型（OB / FVG / LP）、方向（需求/供给）、强度条和填充率（仅 FVG）。
  5. ⑤ 流动性池 — 止损聚集的价格位，按买入/卖出侧分组。显示价格水平、触碰次数和强度。

关键指标怎么读

  • 结构评分 — 0-100 的综合评分。70+ 为健康趋势结构，40-70 为震荡或模糊，低于 40 表示结构破碎或高波动。
  • 溢价 / 折价 — 价格相对于最近主要摆动高/低点的位置。溢价（红色）表示价格在高位，折价（绿色）表示价格在低位。在折价区域找买入机会，在溢价区域找卖出机会。
  • 区域强度 — 0-100% 的进度条。强度高的区域更可能被价格尊重。FVG 还有填充率指标——填充率接近 100% 的缺口吸引力降低。
  • BoS vs CHoCH — BoS（Break of Structure）确认当前趋势延续；CHoCH（Change of Character）暗示趋势可能反转。连续出现 CHoCH 时提高警惕。

常见操作 · 一步步

  1. 按 ⌘K 打开交易对搜索面板，输入或选择你要分析的标的。
  2. 点击时间周期分段按钮切换分析粒度（5m 适合短线，1h/4h 适合趋势判断）。
  3. 看顶部市场状态词和结构评分，判断当前是否适合交易。
  4. 滚动事件时间线，了解价格从过去到现在经历了哪些结构事件。
  5. 查看活跃区域列表——注意距离现价最近且强度最高的区域，这些是最可能的反转/反应位。
  6. 检查流动性池位置，了解止损聚集在哪里——这些通常是价格的"磁石"。

幕后

页面数据来自 /api/structure/market-view BFF 端点，后端根据 K 线数据计算市场状态、结构评分、事件序列、区域和流动性池。ViewModel（MarketStructureViewModel）调用 APIMarketStructure 获取数据。遵循 Redis → 服务计算 → mock 数据的回退模式。区域识别算法基于摆动检测和 SMC 规则引擎。
