公允价值缺口 (FVG)

概念 · 05 / 10 · 5 分钟

FVG = 三根蜡烛形成的"价格真空",市场往往会回来"填补"。
"A Fair Value Gap (FVG) forms when buying or selling pressure is so intense that consecutive candles do not overlap — creating a price window where no trading occurred. Markets have a tendency to return and fill these gaps."— DailyPriceAction[1]

为什么重要

Fair Value Gap (FVG) 是 SMC 体系中最直观也最容易被新人掌握的概念。它的原理很简单：当价格移动太快，相邻的 K 线之间出现了"空隙"——三根 K 线中，中间那根的影线没有与前一根的实体重叠，这个空隙就是 FVG。

为什么会形成 FVG？因为买盘（或卖盘）太强了，价格跳跃式前进，中间有些价位根本来不及成交。这些"没成交"的价位就像没有填实的坑，市场天生有一种"填补"的倾向——价格会回来在这些未成交的价位上寻找流动性。

FVG 既可以作为入场依据（等价格回补到缺口区域时入场），也可以作为止损参考（把止损放在 FVG 的另一侧）。

怎么识别

  1. 找到三根连续的 K 线——从左到右标记为 1、2、3。
  2. 检查 K 线 1 的最高价和 K 线 3 的最低价——如果 K 线 1 的最高价 < K 线 3 的最低价（上升趋势），或者 K 线 1 的最低价 > K 线 3 的最高价（下降趋势），中间就是 FVG。
  3. 标记缺口区域——缺口的上边界是 K 线 1 的最高价（上升趋势），下边界是 K 线 3 的最低价。或者反过来。
  4. 等待价格回补——价格大概率会回到这个区域。当价格进入 FVG 区域并出现反转信号时，可以考虑入场。

注意：FVG 的可靠性随时间推移下降。新形成的 FVG（3-5 根 K 线内）远比 20 根 K 线之前的 FVG 有意义。

常见坑

  • FVG 不等于"一定会回补"。很多 FVG 永远不会被填满，尤其是在非常强趋势的市场中。不要死等一个 FVG 回补。
  • 低时间周期 FVG 太多。5 分钟图上到处都是 FVG。如果每个都关注，你会疯掉。只关注高时间周期（1 小时以上）的 FVG。
  • 忽略成交量。FVG + 高成交量 = 更有意义的缺口。如果 FVG 是在低成交量下形成的，它的"吸引力"弱得多。
  • 把 FVG 当精确止损位。FVG 是一个区域，不是一个精确的点。把止损放在 FVG 边界刚好超出一点的位置，而不是边界上。
