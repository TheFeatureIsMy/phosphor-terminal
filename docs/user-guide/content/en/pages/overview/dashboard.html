Dashboard

The Dashboard is AlphaLoop's trading operation control tower — a single screen showing your account, strategies, positions, risk, decisions, and alerts.

Layout

The Dashboard uses a Bento grid layout, from top to bottom:

Global Status Bar

A pinned infrastructure status bar showing system state, Freqtrade latency, Redis latency, exchange connection, and system-level reason codes.

Account Overview (Hero Card)

Left side displays total equity (animated counter) and 24h change. Right side shows four metrics: Today P&L, Week P&L, Max Drawdown, and Sharpe Ratio.

Suggested Actions

Context-aware quick action buttons the system recommends based on current state — deploy a strategy, review signals, tighten stops. Up to 3 shown.

Three-Column Metrics

  • Strategy Runtime — Running strategies, positions, pending orders, reconciling count
  • Live Readiness — Current readiness lamp (Live Ready / Paper Only / Risk Locked) with gate pass status
  • Global Risk — Risk state, daily/weekly loss remaining gauges

Position Risk Table

Full-width table of open positions: symbol, direction (green/red stripe), size, entry, P&L, risk level, and reason codes.

Decision Feed + Alert Timeline

Side-by-side columns:

  • Recent Decisions — Latest trading decisions with time, symbol, action (execute/hold/reduce/reject), and reason codes
  • Alert Timeline — Chronological system and trading alerts at three severity levels (info/warning/error)

Emergency Action Bar

A fixed bottom bar with "Halt All Trading" button. Requires confirmation before stopping all strategies and canceling pending orders.

Data Refresh

The Dashboard auto-refreshes every 30 seconds via a single aggregated BFF endpoint for data consistency.
