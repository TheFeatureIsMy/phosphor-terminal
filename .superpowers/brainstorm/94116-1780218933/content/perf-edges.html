1. ViewportCuller — 视口裁剪器（新文件）

ViewportCuller.swift 核心逻辑
// 可见性计算 —— 每帧调用，复杂度 O(n)
struct ViewportCuller {
  let padding: CGFloat = 200 // 视口外扩缓冲区

  func visibleNodes(_ nodes: [CanvasNode], viewport: ViewportState, canvasSize: CGSize) → [CanvasNode] {
    // 1. 计算世界坐标系下的可视矩形
    let visibleRect = worldVisibleRect(viewport, canvasSize)
    // 2. 过滤在矩形内（含缓冲区）的节点
    return nodes.filter { node in
      visibleRect.intersects(node.frameWithPadding(padding))
    }
  }
}
StrategyCanvasTab 中使用：
let visible = culler.visibleNodes(vm.graph.nodes, viewport: vm.viewport, canvasSize: geo.size)
ForEach(visible) { node in NodeDragWrapper(...) } ← 仅渲染可见节点
注意：选中节点始终渲染（即使在视口外），通过 selectedNodeIds 补入 visible 集合

4. CanvasEdges 重写 — 粒子动画

CanvasEdges.swift — 增强渲染
新增渲染特性：
• 默认 2px 线宽，选中态 3px + glow
• 数据流方向粒子（每 80px 一个点，沿路径运动）
• 粒子颜色 = dataType.color，透明度 0.6
• 粒子动画用 TimelineView 驱动（不是 Timer）
• 禁用态边线变虚线 + 灰色
性能策略：
• 仅绘制可视边（两端节点至少一个在视口内）
• 粒子在 Canvas 中批量绘制，避免多次 stroke 调用
• scale < 0.3 时粒子不可见（zoom out 时不浪费）
• TimelineView 的 animation 在无粒子时自动暂停
