3. 交互系统增强

Clipboard · Snap · Multi-select · Search
📋 复制粘贴 ClipboardManager
⌘C → 序列化选中节点+边到内部剪贴板（NSPasteboard）
⌘V → 反序列化 + 新 UUID + 偏移 (50,50) 放置
⌘D → duplicate（⌘C + ⌘V 一步完成）
剪贴板数据格式：JSON（GraphSerializer 的节点子集 + 边子集，不含 viewport）
📐 对齐吸附 SnapEngine
• 吸附阈值：8px
• 吸附目标：其他节点边缘（上/下/左/右/中心）
• 吸附时显示 SnapGuidesView（虚线辅助线）
• 网格吸附可选（20px grid，Shift 键切换）
• 只在拖拽结束时计算（不在 onChanged 中）
👥 多选操作增强
• 框选：SelectionRect 接入多节点选中 + 多节点拖拽
• 多节点拖拽：所有选中节点同步移动（保持相对位置）
• ⇧ 点击：范围选择（追加到选中集）
• ⌘A：全选
• GroupBox 联动：选中组内节点 → 可一起移动/删除
🔍 画布内搜索 CanvasSearchOverlay
• ⌘F 打开搜索浮层（悬浮在画布上方中央）
• 输入关键词 → 匹配的节点高亮 + 视口动画平移至第一个结果
• ⌘G / ⇧⌘G 跳转到下一个/上一个匹配
• 无匹配时搜索框变红 + 微抖动
